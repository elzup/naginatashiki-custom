# naginatashiki-custom

[薙刀式](https://oookaworks.seesaa.net/article/498149829.html) v16 公式 Karabiner-Elements 設定 (`a296a39 clone origin`) に独自カスタマイズを加えたもの。

## ファイル

- `karabiner-elements_a-base.json` — 本体ルール (大半のカナ・編集モード等)
- `karabiner-elements_b1-shift-kana.json` — シフトカナ拡張
- `karabiner-elements_b2-enter-shift.json` — Enter 同時押しシフト

## オリジナルからの差分

### 基本面に `<key>+o` 同時押しで小書き「ぇ」入力を追加

オリジナルでは「ぇ」拗音 (しぇ・じぇ・ちぇ等) は **右半/左半/濁シフト経由の3キーシンクロ** が必要だった。`ゃゅょ` が基本面で 2キーシンクロ (`r+h`=しゃ, `r+p`=しゅ, `r+i`=しょ) で打てるのに対し、ぇだけ手数が多い。

そこで `<子音キー> + o` (`o` は基本面の「え」) の 2キーシンクロで打てるルールを追加した。

#### 子音 + え

| 同時押し | 出力 |
|---|---|
| `s + o` | みぇ |
| `e + o` | りぇ |
| `r + o` | しぇ |
| `w + o` | きぇ |
| `d + o` | にぇ |
| `g + o` | ちぇ |
| `x + o` | ひぇ |

#### 母音/特例 + え

| 同時押し | 出力 | 備考 |
|---|---|---|
| `k + o` | いぇ | `k` は基本面で「い」 |
| `l + o` | うぇ | `l` は基本面で「う」 |
| `; + o` | ふぇ | `;` は基本面で「ー」だが、ーぇ は使わないので上書き |

### 触っていないもの (オリジナルのまま)

- 濁音/半濁音 + ぇ (じぇ/ぢぇ/ヴぇ/ぐぇ) — 既に 3キーシンクロで登録済み (例: `j+r+o`=じぇ)
  - 2キー化は `r+o`=しぇ と衝突するため不可。`ょ vs じょ` と同じ構造で 3キー必須
- ぎぇ・びぇ・ぴぇ・でぇ・ぜぇ・ぶぇ等のレア音 — 未登録 (元々の薙刀式仕様)

## 適用方法

Karabiner-Elements の `Complex Modifications` に各 JSON を import する。
